ERRAI Framework

Enterprise Applications need to:

- know user info and application data

- be access restricted

- be customized (Splash Page, Wait Panel, ...)

- have common features (search, async callback, ...)

IOC to obtain info and application data

@Inject
BootstrapData bootstrapData;
...
Window.alert(bootstrapData.getUserInfo().getUsername());

Errai Security to restrict access to pages

Through Rescricted Access Annotation over:

- REST resources

- UI elements

@GET
@Path("/{id}")
@Produces(MediaType.APPLICATION_JSON)
@RestrictedAccess(roles = "gwtcon-admin")
public Customer find(@PathParam("id") long id);

Intercerptors to handle Async Callback

Through interceptors over REST resources

@GET
@Path("/{id}")
@Produces(MediaType.APPLICATION_JSON)
@InterceptedCall(RequestInterceptor.class)
public Customer find(@PathParam("id") long id);
